Tudo Sobre HTML

HTML (abreviação para a expressão inglesa HyperText Markup Language, que significa Linguagem de Marcação de Hipertexto) é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores. - wikipedia.org - acesso 09/2016

Tecnologia - Realidade Aumentada

De uma forma simples, Realidade Aumentada é uma tecnologia que permite que o mundo virtual seja misturada ao real, possibilitando maior interação e abrindo uma nova dimensão na maneira como nós executamos tarefas.

Acessibilidade

Acessibilidade na web significa que as pessoas com deficiência podem usar a web. Mais especificamente, a acessibilidae na web significa que as pessoas com deficiencia podem perceber, entender, navegar, interagir e contribuir para a web.
